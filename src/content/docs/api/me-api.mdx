---
title: Me
---

All informations about your session.

**Endpoints**

```text
GET /api/me
GET /api/me/teams
GET /api/me/websites
```

---

## GET /api/me

Get informations based on your auth token.

**Sample response**

```json
{
  "token": "xxxxxxxxxxxxxxx",
  "authKey": "auth:xxxxxxxxxxxxxxx",
  "shareToken": null,
  "user": {
    "id": "7c78d7e8-cb8e-4c96-9bfc-708379297dda",
    "username": "member",
    "role": "user",
    "createdAt": "2025-10-08T18:03:19.823Z",
    "isAdmin": false
  }
}
```
